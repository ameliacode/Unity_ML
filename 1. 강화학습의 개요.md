# 강화학습의 개요



## 기계 학습이란?

정의: 기계가 일일이 코드로 명시하지 않은 동작을 데이터로부터 학습하여 실행할 수 있도록 하는 알고리즘을 개발하는 연구 분야

#### 기계 학습 알고리즘

* Decision Tree
* Random Forest
* Support Vector Machine
* Artifical Neural Networks

#### 기계 학습 유형

* Supervised Learning: Label이 있는 Train Dataset에서 Feature을 통해 Predict → Minimize errors 
  * Regression
  * Classification
* Unsupervised Learning: 정답이 따로 존재 X, Clustering 문제 해결, 데이터 기반
* Reinforcement Learning:: 환경과 상호작용 → 보상

### 강화학습의 기본 용어

Markov Decision Process: 

* Agent: 의사결정 역할
* Environment: 에이전트에게 정보를 주는 역할
* Observation: 환경에서 제공해주는 정보, 시각적 관측, 수치적 관측
* State: 에이전트는 상태를 기반으로 의사결정을 한다.
* Action: 에이전트가 의사결정을 통해 취할 수 있는 행동
* Reward function: 에이전트가 특정 상태에서 특정 행동으로 보상을 받을 때 학습을 진행
* Discount factor: 효율적으로 보완
* State transition probability: 상태 s에서 행동 a를 했을 때 다음 상태가 s가 될 확률
* Policy: 특정 상태에서 취할 수 있는 행동을 선택할 확률 분포



### 강화학습 기본 원리

⚠ 자세한 설명은 책 참고 ⚠

강화학습 → 경험을 통해 보상을 최대화하는 의사결정 전략을 학습

* 감가율(Discount factor): 기록하는 현 정보들을 보완
* Exploration: 다양한 경험을 통해 Agent의 행동을 결정
* Exploitation: Agent의 행동을 결정하는 기법, 대표적으로 탐욕적 방법,  epsilon-greedy(Random action을 취할 확률, epsilon 값이 감소, 무작위 탐색에서 탐욕적 행동으로 확장)
* 가치 함수(Action Value Function): 상태의 반환 값에 대한 기댓값 반환
* 큐 함수(Statue Value Function): 
* 벨만 방정식: 가치 함수는 벨만 방정식을 통해 업데이트 → 현재 상태의 가치 함수와 다음 상태 가치 함수로 표현한, 기댓값
* 가치 기반 강화학습: 벨만 최적 방정식을 통해 시간차 예측 → 최적의 큐 함수, 대표적으로 DQN
* 정책 기반 강화학습: 정책 경사를 통해 학습 by 목적 함수 → 최적의 정책 업데이트,  대표적으로 REINFORCE, A2C
